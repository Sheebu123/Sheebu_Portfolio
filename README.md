# Sheebu Tiwari Portfolio

Developer portfolio built with Next.js App Router, Tailwind CSS, Framer Motion, and EmailJS.

## Stack

- Next.js
- Tailwind CSS
- Framer Motion
- EmailJS
- TypeScript

## Run locally

```bash
npm install
npm run dev
```

## Environment variables

Create `.env.local` from `.env.example` and set:

```bash
NEXT_PUBLIC_EMAILJS_SERVICE_ID=
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=
```
